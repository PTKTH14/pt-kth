💾 ดาวน์โหลดข้อมูลสำรอง Firebase

เก็บ Checklist ผลประเมิน รายงานส่วนตัว รายการผู้ไม่ได้ประเมิน ประวัติการใช้งาน และรายชื่อบุคลากรไว้ในไฟล์ JSON หนึ่งไฟล์

ยังไม่เคยดาวน์โหลดในอุปกรณ์นี้
ไฟล์สำรองมีข้อมูลภายในแผนก กรุณาเก็บในพื้นที่ที่จำกัดสิทธิ์ และระบบจะไม่รวมรหัสผ่านแอดมินหรือ PIN หัวหน้างาน

♻️ กู้คืนข้อมูลจากไฟล์

ระบบจะเติมรายการที่หายและเขียนทับรายการเดิมที่มี ID ตรงกัน โดยไม่ลบรายการอื่นที่มีอยู่ใน Firebase
